# Licences

## Audio samples

The cello samples in the `samples/` folder are from the
**Sonatina Symphonic Orchestra** library, created by Mattias Westlund.

- Source: https://github.com/peastman/sso
- Licence: Creative Commons Sampling Plus 1.0
  https://creativecommons.org/licenses/sampling+/1.0/

The samples have been modified for use in Lungful: pitch-shifting is applied
at runtime via the Web Audio API playbackRate parameter, and the sustained
sample is looped within a defined sustain window.

Files used:
- `cello-c2.flac` — bowed cello C2, used for inhale/exhale glide
- `cello-piz-rr1-c2.flac` — pizzicato cello C2, used for hold after exhale
- `cello-piz-rr1-g2.flac` — pizzicato cello G2, used for hold after inhale
